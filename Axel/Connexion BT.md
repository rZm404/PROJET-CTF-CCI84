# Titre : BT connexion
## Description
Un appareil bluetooth semble s'être connecté à mon ordinateur, seriez vous retrouver le nom de l'appareil qui s'est connecté ?

Le flag ressemblera au suivant : `NHM2I{XXX_XXXXXX_XXX}`
## Résolution
![image](https://user-images.githubusercontent.com/74879480/214540818-6508533c-61c7-4f8d-8077-a03946ff4d34.png)

Le flag est donc : `NHM2I{Galaxy_Tab_S5e}`
